# Tech Stack

## Frontend Framework
- **Astro (v5.16.4):** Used as the primary web framework for building a fast, content-focused website with partial hydration.

## Programming Languages
- **TypeScript/JavaScript:** Utilized for both build-time logic and client-side interactivity, ensuring type safety and modern syntax.

## Styling & Design
- **Tailwind CSS (v4.1.17):** A utility-first CSS framework for rapid UI development and consistent styling.
- **GSAP (v3.14.2):** High-performance animations for engaging user interactions.
- **Lenis (v1.3.16):** Smooth scrolling library to enhance the overall browsing experience.

## Client-side Logic
- **Alpine.js (v3.15.2):** A lightweight JavaScript framework for handling simple client-side reactive behavior and DOM manipulation.

## Utilities
- **Marked:** Used for parsing and rendering Markdown content within the application.
